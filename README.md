# patterni

ну короче это лаба по билдеру на с++

## что тут есть

- класс `Burger` — сам бургер с ингредиентами
- `BurgerBuilder` — абстрактный класс для сборки
- `ClassicBurger`, `VeganBurger`, `SpicyBurgerBuilder` — три вида бургеров
- `Cashier` — типа кассир, который всё это собирает

## как запустить

```
g++ main.cpp -o burger
./burger
```

ну или через свою идею, если лень в консоли ковыряться

## что выводит

просто список ингредиентов для каждого бургера, типа:
```
Состав бургера: 
	 - Булочка: х1 Классическая с кунжутом
	 - Котлета: х1 Говяжья котлета
   ...
```

## если хочешь добавить свой бургер

1. создаёшь класс от `BurgerBuilder`
2. переопределяешь все `build*` методы
3. добавляешь в `main` и готово
